# Color Scheme - Consistent Throughout

## Primary Colors
- **Primary Blue**: `#2563eb` - Used for all buttons, badges, links
- **Primary Blue Hover**: `#1d4ed8` - Used for button hover states
- **Light Blue**: `#3b82f6` - Secondary blue shade
- **Blue Background**: `#eff6ff` - Light blue background for highlights

## Neutrals (Consistent Everywhere)
- **White**: `#ffffff` - Main background, cards
- **Off-White**: `#f8fafc` - Alternate backgrounds
- **Light Gray**: `#f1f5f9` - Section backgrounds
- **Gray Borders**: `#e2e8f0` - All borders
- **Text Gray**: `#475569` - Body text, navigation
- **Darkest**: `#0f172a` - Headings, footer background

## Section Backgrounds (No Changes on Scroll)
1. **Header**: Always white `#ffffff`
2. **Hero Section**: White to off-white gradient
3. **Services Section**: Pure white `#ffffff`
4. **Testimonials Section**: Light gray `#f1f5f9`
5. **Contact Form Section**: Pure white `#ffffff`
6. **Final CTA**: Primary blue `#2563eb`
7. **Footer**: Darkest gray `#0f172a` (for contrast)

## Interactive Elements
- **All Buttons**: Primary blue `#2563eb`
- **Button Hover**: Primary blue hover `#1d4ed8`
- **Links**: Primary blue `#2563eb`
- **Link Hover**: Primary blue hover `#1d4ed8`

## Consistency Rules
✅ NO color changes on scroll
✅ NO background changes on scroll
✅ Same shadow style everywhere
✅ Same border radius everywhere (8px for small, 10-12px for large)
✅ Same transition speed (0.2s)
✅ White cards with light borders throughout

## What We Removed
❌ Dynamic header color changes
❌ Section background color variations
❌ Yellow/gold colors
❌ Dark theme sections
❌ Inconsistent button styles
❌ Pulse animations
❌ Floating animations
